# 📖 < COCOA WEBTOON > 서비스 소개
이 프로젝트는 요일 별 연재되는 웹툰들을 사용자가 회원가입, 로그인, 캐시충전, 웹툰구매, 댓글, 좋아요 등의 기능들과 함께 즐길 수 있도록 구현한 웹 서비스입니다.
